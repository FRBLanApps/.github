## Who We Are

FRBLanApps is a student team focused on Flutter development and Root module development.

We care about three things:

- Useful products over flashy demos
- Continuous iteration over one-shot perfection
- Readable, maintainable code over short-term hacks

## What We Build

- PiliSuper: developing and improving the core experience
- Flowgu: continuous optimization and feature polishing
- UnTrickyStore: in active development

## Tech Focus

- Languages: Dart, Python, Rust
- Mobile and UI: Flutter
- Engineering style: lightweight process, fast feedback, gradual hardening

## Current Stage

We are an early-stage team, so we keep our workflow simple and practical:

- Small scope, fast delivery
- Prioritize shipping over over-design
- Improve docs and standards step by step as projects mature

## Join Us

If you want to contribute, the easiest way is:

1. Open an Issue to report bugs or propose ideas
2. Pick a task and submit a Pull Request
3. Discuss improvements in Issues or PR threads
